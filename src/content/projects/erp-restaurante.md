## Desafio

Desenvolvi este sistema de gestão (ERP) para restaurantes como um desafio pessoal de engenharia: criar uma aplicação web complexa que funcionasse inteiramente no navegador, sem backend.

## Arquitetura

A arquitetura é **local-first** e utiliza **IndexedDB** para persistir e gerenciar gigabytes de dados desde o histórico financeiro até o fluxo de pedidos em tempo real.

## Módulos

O projeto é totalmente modular, integrando frentes de caixa (**PDV**), monitores de cozinha, gestão de delivery e contabilidade.
